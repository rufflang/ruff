# v0.12.0 LSP Completion Engine (Builtins/Variables/Functions)

Date: 2026-04-29
Scope: Active-cycle roadmap execution for highest-priority incomplete v0.12.0 item (Language Server Protocol)

## What was implemented

- Added completion engine module: `src/lsp_completion.rs`
  - Completion item kinds: builtin, function, variable
  - Cursor-prefix extraction from source using 1-based line/column inputs
  - Symbol collection from parsed Ruff statements:
    - user-defined functions (`func`)
    - user-defined variables (`let`, `const`, `for` variables, destructuring bindings)
  - Builtin inclusion via `Interpreter::get_builtin_names()`
  - Name-collision precedence: user symbols outrank builtin symbols

- Added CLI command:
  - `ruff lsp-complete <file> --line <N> --column <N> [--json]`
  - Plain output: `<label>\t<kind>` per line
  - JSON output for editor integration

## Verification

Commands run:

1. `cargo test lsp_completion -- --nocapture`
- Result: PASS
- Coverage validated:
  - prefix extraction behavior
  - builtin/function/variable candidate coverage
  - user-over-builtin precedence on collisions

- Result: PASS
